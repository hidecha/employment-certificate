# 就労証明書解析スキル (analyze_shuroushomei)

日本の各自治体の就労証明書Excelフォーマットを解析し、全テキスト入力フィールドとチェックボックスのセルアドレスマッピングを作成する。

## クイックスタート

解析スクリプトの実行:

```bash
PYTHONIOENCODING=utf-8 python skill/scripts/analyze_shuroushomei.py <ファイル名>.xlsx
```

JSON形式で出力:

```bash
PYTHONIOENCODING=utf-8 python skill/scripts/analyze_shuroushomei.py <ファイル名>.xlsx --json
```

### スクリプトフラグ

| フラグ | 説明 |
|--------|------|
| `--json` | JSON形式で結果を出力。`{municipality, text, checkbox}` を返す |

スクリプトの自動処理:
- 自治体名をExcelの内容から推測（ファイル名ではなく）
- セル構造と結合セルを解析
- 全テキスト入力フィールドのアドレスを特定
- 全チェックボックス(□)の位置をラベル付きで検出

## 出力形式 (JSON)

```json
{
  "municipality": "○○市",
  "text": {
```

## 技術メモ

- openpyxl実行時は `PYTHONIOENCODING=utf-8` を必ず指定する
- 結合セルの左上座標が入力セルのアドレスとなる
- 自治体名はExcel内容から推測（例: ヘッダーの「○○市長」）
- チェックボックス検出は `_detect_checkbox_label()` で□セルの右隣ラベルを取得、`_detect_row_context()` で行コンテキストを付与
- チェックボックスの形式は自治体により異なる（□記号 vs ☐/☑/✓ — `_repair_checkboxes()` で代替マーカーも検出）
- `analyze_certificate()` はセクション別パーサー (`_parse_*`) を順次呼び出す構成。新セクション追加時は対応する `_parse_*` 関数を作成し、`_section_parsers` リストに登録
- 休業系セクション (No.8/9) は共通関数 `_parse_leave_period()` を使用
- 期待セクション定義 (`_EXPECTED_TEXT_SECTIONS`, `_EXPECTED_CB_SECTIONS`) は本スクリプトで定義し、main.py から `importlib` でインポート（単一ソース）
